h1. Vim

Venho lendo que não dá para depender de IDEs há algum tempo. E, que é possível ser mais produtivo com um editor de textos poderoso.
Então, aproveitando minha atual entrada no mundo Ruby e o fato do Akita, que é quem eu leio sobre Ruby e usa VIM, resolvi me inspirar na idéia de "versionar meu diretório ~/.vim usando o Git":http://addisu.taddese.com/blog/using-github-and-pathogen-for-your-vim-config-files.

Para modificar as configurações entre no Vim e digite:

:edit $MYVIMRC

h1. Bundled Plugins

 * "Pathogen":https://github.com/tpope/vim-pathogen - "Como instalar os plugins no VIM encapsulados em um diretório.":http://gustavodutra.com/post/409/pathogenvim-uma-alternativa-para-quem-utiliza-plugins-em-repositorios-remotos, ou "Como gerenciar os arquivos de configuração do VIM"

  * "Command-T":http://git.wincent.com/command-t.git/blob_plain/HEAD:/README.txt - Plugin que emula a abertura de arquivos do TextMate usando Command-T. No caso do VIM o atalho é <Leader>t, onde <Leader> costuma ser o \.
